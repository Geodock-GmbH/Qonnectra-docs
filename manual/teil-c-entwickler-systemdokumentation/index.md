# Teil C - Entwickler- und Systemdokumentation

Dieser Teil des Handbuchs richtet sich ausschließlich an technische Rollen, insbesondere an Entwickler, DevOps, GIS-Administratoren sowie Mitglieder der Open-Source-Community.

Teil C beschreibt die **technische Architektur von Qonnectra**, den Systemaufbau, die Installation, das Datenmodell sowie Möglichkeiten zur Erweiterung und Weiterentwicklung des Systems.

Die Inhalte dieses Teils sind **nicht erforderlich für den fachlichen Betrieb** von Qonnectra und setzen technisches Vorwissen voraus. Änderungen auf dieser Ebene können Auswirkungen auf den gesamten Systembetrieb haben und sollten nur durch entsprechend autorisierte Personen erfolgen.
